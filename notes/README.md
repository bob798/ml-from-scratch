## 写笔记的规则

1. **自包含**：文档本身可独立阅读，不依赖原始对话或外部上下文
2. **一篇一题**：每篇只解决一个核心问题，过长就拆分
3. **先扁平后分子目录**：每个子目录文档 5+ 篇后再考虑细分
4. **结论先行**：文档开头给出核心结论或推荐，再展开论证
5. **业务语言优先**：能用业务语言讲清楚的就不用术语
