## 斜率：就是「陡不陡」

斜率回答一个朴素问题：**走一步，升多高？**

```
斜率 = 上升量 / 水平移动量 = Δy / Δx
```

（Δ 念 delta，就是「变化量」。）

- 斜率大 = 陡；斜率小 = 缓；斜率 0 = 平地；斜率负 = 下坡

直线很简单，每一点一样陡，整条线一个斜率：

```
两点 (1,3) 和 (3,7)：斜率 = (7−3)/(3−1) = 2
意思：x 每走 1，y 升 2
```

## 问题：曲线怎么办

y = x² 这种曲线，每个地方陡的程度都不一样。「曲线在某一点的斜率」是什么意思？一个点没有「走一步」，怎么算 Δy/Δx？

**这就是求导要解决的唯一问题：求出曲线在某一点的斜率。**
求导（derivative）= 求某一点的斜率。「导数」= 那一点的斜率。

## 怎么求：放大镜思想

核心技巧：**一个点不好算，就在它附近取很近的两点，算这两点连线的斜率，当作这一点的斜率。**

求 y = x² 在 x = 3 处的斜率，两点越靠越近：

```
跨度 1    → 7
跨度 0.1  → 6.1
跨度 0.01 → 6.01
跨度 →0   → 6   ← 这就是 x=3 这一点的真正斜率
```

「把两点无限拉近，看斜率稳定在哪」——这个稳定值就是导数。「无限拉近」数学里叫**极限**，但你已经亲眼看到它收敛到 6 了。

## 代码验证

```python
def f(x): return x**2
x = 3
for h in [1, 0.1, 0.01, 0.001, 0.0001]:
    slope = (f(x + h) - f(x)) / h
    print(f"跨度 {h:<7} → 斜率 {slope}")
# 输出逐步逼近 6.0
```

## 那个公式从哪来

x² 求导得 2x，只是偷懒用的：数学家发现 x² 在任意点 x 的斜率永远是 2x，不用每次跑逼近过程。

- x=3 → 6 / x=5 → 10 / x=0 → 0（谷底，平的）

**导数 f'(x) = 2x 本身又是一个函数**——输入「位置 x」，输出「那一点的斜率」。

## 为什么 ML 死磕这个

训练模型 = 找损失函数的最低点。斜率告诉你往哪走、走多急：

```
斜率为正 → 右边在升 → 往左走
斜率为负 → 右边在降 → 往右走
斜率 ≈ 0 → 到谷底了 → 停
斜率大（陡）→ 离谷底远 → 迈大步
斜率小（缓）→ 快到了   → 迈小步
```

这就是**梯度下降**：站在曲线上，算斜率，朝下坡迈一步，反复迈到谷底。求导就是模型「用脚探路找最低点」的那只脚。

## 验收三问

1. 斜率是什么？（走一步升多高）
2. 求导在求什么？（曲线上某一点的斜率）
3. 怎么求出来的？（取附近两点算斜率，把两点无限拉近，看稳定值）
