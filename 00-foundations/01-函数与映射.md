# 01 · 函数与映射

> ML 的地基。搞懂这一篇，「神经网络是什么」就不再神秘。

## 动机：函数解决什么问题

你有一堆输入，想得到对应的输出，并希望这种「对应关系」是**确定的、可复用的**——同样的输入永远给同样的输出。函数就是把「对应关系」本身变成一个可以研究、可以操作的对象。

**ML 锚点**：你有一堆房子的面积（输入），想预测房价（输出）。你要找的就是那个「面积 → 房价」的对应关系。整个监督学习，就是在找这个函数。

## 依据：别背，整篇只有「2 条根 + 推论」

如果你在硬背这一篇，多半是因为它看起来像十几条要记的事实。其实不是——**整篇只有 2 条需要内化的「根」，其余全是能现场推出来的「叶子」**。能推出来就不用背。

**根 1 · 定义**：函数 = 一种特殊的「关系」，要求每个输入**恰好**对一个输出。
（集合论说法：函数是笛卡尔积 A×B 的一个子集，且 A 里每个 a 只配一个 b。）

**根 2 · 动机**：为什么单挑这种关系？因为要「**确定、可复用**」——同输入同输出，工具才有用（见上面的导航类比）。

**从这两条长出来的推论（不用记，现场推）：**

| 文中结论 | 从哪条根推出来 |
|---|---|
| 为什么必须「唯一输出」 | 根 1 定义里就写着，不是额外规定；去掉它就不叫函数，叫关系 |
| x=y² 为什么不是函数 | x=4 配 ±2，违反「恰好一个」，掉回关系 |
| 神经网络是函数 | 满足「输入定一个输出」即可，根 1 没要求规则必须是公式 |
| 概率分布也是函数 | 把「整个分布」当成单一输出，仍满足唯一性 |
| 复合、反向传播能成立 | 函数能首尾相接（输出当下一个的输入），是定义的直接结果 |

**自检你是不是在背**：盖住右边，只看 2 条根，能不能把每条结论重新讲一遍。能 → 你理解了；卡住 → 是某条根没吃透，回去补根，别去背叶子。

## 是什么

函数 f 是从集合 A（定义域）到集合 B（陪域）的对应，使得 A 里每个元素**恰好**对应 B 里一个元素。记作 `f: A → B`。

**关键转变**：函数**不一定是公式**，它本质是「映射」——一张输入到输出的对照表。Dirichlet 把函数从「算式」解放成「映射」，这一步直接决定你能不能理解神经网络。

## 为什么必须「恰好对应一个输出」

不是死板规定，是这个工具要有用就必须这样。「确定」的意思就是：同样的输入，必须给同样的、唯一的输出。

类比导航：问「从家到公司怎么走」，答「走 A 路线，20 分钟」能用；答「可能 20 分钟也可能 50 分钟，看心情」就没法用。一个输入吐两个答案，你不知道信哪个。

**ML 锚点**：训练好的模型，输入一张猫的图必须给确定答案。同一张图一会儿说猫一会儿说狗，模型就废了。「输出唯一」是模型能用的前提。

## 不是什么（用反例划边界）

- 一个输入对应两个输出 → **不是**函数（如 x = y²，x=4 时 y 可能是 +2 或 −2）
- 必须是公式 → **不是**，查找表、一段代码、训练好的神经网络都是函数
- 必须连续/光滑 → **不是**，阶梯函数也是函数

## 函数本质是「映射」（最该想通的一点）

函数 = 一张**对照表**：左边输入，右边输出，左边每个恰好指向右边一个。

```
输入 → 输出
 1  →  2
 2  →  5
 3  →  10
```

这三行就是一个函数。它碰巧能写成 x²+1，但就算写不出公式照样是函数：

```
"猫的照片"  → "猫"
"狗的照片"  → "狗"
```

没有任何算式能描述，但它完全合法。**函数 = 一套确定的指向规则**，规则是用公式写的、查找表存的、还是几亿个神经网络参数算的，都无所谓。

**ML 锚点**：这就是为什么神经网络是函数。它没有简洁公式，是一张靠几亿参数算出来的超级大对照表「图像 → 标签」。接受「函数=映射而非公式」，「神经网络是什么」就不神秘了——它就是个写不出公式的函数。

## 复合：把函数串成生产线

复合 = 把一个函数的输出，喂给另一个函数当输入。

设 g(x) = 2x + 1，f(x) = x²，复合 f(g(x))：

```
x = 3
 ↓  先过 g：g(3) = 2×3+1 = 7
 ↓  g 的输出 7 喂给 f：f(7) = 49
结果 = 49
```

**ML 锚点（命门级）**：神经网络就是一堆函数的复合。

```
输入 x → 第1层 f₁ → 第2层 f₂ → 第3层 f₃ → 输出
```

整个网络 = f₃(f₂(f₁(x)))。而**反向传播**就是给这条复合生产线求导（链式法则）。搞懂复合，反向传播的骨架就在手里了。

## 「不确定的对应」叫什么

x = y²（一个输入对多个输出）不是函数，它叫**关系（relation）**。函数是关系的特例——加了「输出唯一」限制的关系。

```
关系：任意对应，一个输入可对多个输出
  └─ 函数：每个输入恰好对一个输出
```

要让「一对多」可用的几种办法：
1. 当方程/曲线看待（x=y² 是合法曲线，只是不是函数）
2. 拆成多个函数（y=+√x 和 y=−√x）
3. 输出升级成**集合**或**概率分布**

**ML 锚点（重点）**：现实里「一个输入对多个合理输出」很常见（同一句中文有多种翻译）。模型不输出确定答案，而是输出**概率分布**「猫 70%、狗 25%」。妙处：输出「一个分布」本身是唯一确定的，所以它**仍是函数**。大模型本质就是个函数：输入上文，输出「下一个词的概率分布」。

## 验收三问

1. 用映射（不用公式）一句话说清函数是什么？为什么 x=y² 不是函数？
2. 复合 f(g(x)) 怎么算？神经网络和复合什么关系？
3. 「一对多」的对应叫什么？ML 用什么技巧把它装回函数框子？

## 出处与延伸（怎么验证这些不是我编的）

这篇每条结论都能在标准教材或原始论文里查到。想确认对错，按下面去对：

**函数 = 映射 / 任意对应（根 1 的来源）**
- 现代「任意对应、不要求有公式」的函数观，主流归功于 **Dirichlet（1837）**；1930 年代 **Bourbaki** 用集合论（A×B 的子集）把它精确化。注意这是「主流归功」，历史细节学界有争议。
- 教材验证：Halmos《Naive Set Theory》（函数 = 关系的特例）、Rudin《数学分析原理》第 1 章、Rosen《离散数学及其应用》「函数」一章。
- 综述：[Evolution of the Function Concept: A Brief Survey](https://www.researchgate.net/publication/251442942_Evolution_of_the_Function_Concept_A_Brief_Survey)

**神经网络 = 函数 / 函数复合，且能逼近任意连续函数**
- 严格依据是**万能逼近定理**：**Cybenko（1989，用 sigmoid 激活）**、**Hornik（1991，推广到更一般的激活）**。它给出「神经网络是个够强的函数」的数学保证。
- 教材验证：Goodfellow、Bengio、Courville《Deep Learning》(2016) 第 6 章。
- 入门笔记：[A note on Cybenko's Universal Approximation Theorem](https://arxiv.org/html/2508.18893v1)

**概率分布作为输出（最后一节 softmax）**
- 《Deep Learning》第 6.2 节：分类网络用 softmax 输出「各类别的概率分布」，输出「整个分布」本身是确定的，所以仍是函数。

> 用法：遇到任何一句「为什么是这样」的疑问，先回到上面对应的根和出处，而不是把结论当口诀背。
