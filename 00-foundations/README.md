# 00 · Foundations（基础:从函数到反向传播）

机器学习的数学地基。一句话脊柱:**ML = 从数据里找一个函数。**

## 全景:四篇怎么咬合成「训练一个模型」

别把四篇当四个孤立知识点——它们是一台机器的四个零件,合起来就是「训练」这一个动作:

```
                  ┌─────────────────────────────────────────────┐
                  │            训练 = 把 loss 压到最小            │
                  └─────────────────────────────────────────────┘

  [01 函数]      模型就是一个函数  f(x; w)        ← 要学的「对应关系」,w 是旋钮
       │                                            神经网络 = 一堆函数复合
       ▼
  [04 MSE]       给 f 的输出打分   loss = 差多少    ← 有了分数,才有优化目标
       │
       ▼
  [02 导数]      loss 对每个旋钮 w 多敏感?斜率      ← 往哪拧、拧多急
       │
       ▼
  [03 链式+下坡] 复合求导(反向传播)拿到梯度          ← 把"责任"倒推回每个 w
                 w ← w − lr × 梯度 (沿斜率下坡)
       │
       └──⟳ 循环:前向算 loss → 反向算梯度 → 下坡更新 w → loss 变小 → 再来
```

**一句话串起来**:模型是个**函数**(01),用 **MSE**(04)给它打分,靠**导数**(02)知道每个参数该往哪调,用**链式法则做反向传播 + 梯度下降**(03)一步步把分数压低——循环到谷底,模型就「学会」了。
